Shifting Mausoleum

Boss Rush 2024 Game Jam

This game jam seemed like a good place to make a game with some friends over a month. On the genre of boss rushes, with the revealed theme of "exchange", this was a good opportunity to make some features I've wanted to make for a long time. And so my tasks for the game included a lot of things: environment generation, spellcasting and spells, animation and locomotion systems,

Results
