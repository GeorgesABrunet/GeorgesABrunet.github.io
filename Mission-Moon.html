Mission: 13151514

Game Off 2020 Game Jam

This month long project was a very big learning experience. Mission Moon is a game created by me and Olivia Maciejewska. The game jam required that we use Github for version control and to make all of our code public and available to everyone! I'm a strong believer in transparent development so I personally loved this idea. The original game was to be a resource management survival, where the player had to balance the resources produced to redistribute them to multiple aspects of the ship in order to make it to the moon. Upon playing the finished version, we realized the game was missing an "umph", or the most important element, fun. We overhauled it with 4 days remaining to make it a survival shooter! Olivia designed a neat enemy AI with solid pathfinding and combat elements as well as directives. I implemented a combat system for the player, dynamic layered animation schemes for all entities in game, a wave system to distribute the enemies at random locations across the ship, object and player health system, win and lose states, world-space indicators, a dynamic-rendering pipeline, as well as all the shaders and art assets.

Results

Ratings are underway until December 21 2020, so I will update this page when it is complete! I loved making the assets for this game as well as the effects and programming the aspects of the game. I plan on creating a game like this in 3D soon and will also release it on itch.io. Check the Game Jams page in projects for more!
