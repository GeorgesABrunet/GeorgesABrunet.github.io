Bolder Games Released Titles

A collection of the company's released titles

Bolder Games has a cool portfolio of games. Throughout my time at Bolder Games, I've worked as a software engineer with generalist responsibilities. Here are the titles I've had the pleasure to work on that have been released or re-released!

Deep Space Scoundrel

Implemented enemy spawning logic, enemy movement systems, enemy creation tools, player ship movement, player ship configuration and firing system, garage system, player locomotion system, and garage UI system. All systems were authored in C++, but are extended to blueprint.

Starballs

Re-release, I optimized the codebase to improve performance by refactoring physics, gameplay systems. Refactored UI systems and rendering pipeline for release on the Nintendo Switch and Steam for Windows.

The Adventures of Spunk Dodgers & Splat

Re-release. I optimized the codebase by refactoring gameplay systems as well as the UI to deploy on the Nintendo Switch.

Roboshark Rampage

Re-release. Refactored enemy spawning and update logic to increase performance for both CPU and memory. Refactored UI systems and rendering pipeline for release on the Nintendo Switch.
