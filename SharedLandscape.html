Shared Landscape System

Getting your landscape materials to layer nicely with one another in Unreal Engine 4

If you followed the Persistent Trails tutorial, you should have the preliminary parts of this built already. As to not repeat steps, this tutorial builds directly on top of that system. So, if you don't have that setup already, just find the Persistent Trails tutorial on my portfolio page and do it so you're up to speed. A nice note to make is that you won't need to create any new materials! Yay. We're just upgrading things here. Now, that we have persistent trails, you'll notice that the effect currently works the same for all materials. This is by its current design, but if you want to have a system to separate materials that leave trails from those that do not (or leave a different type of trail), then you'd want to implement a system like the one shown in this tutorial. So from the point where we ended the persistent trails tutorial, we're only going to make a handful of additions and edits to the landscape material to make a system for our landscape layers.
